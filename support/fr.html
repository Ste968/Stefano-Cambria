Problèmes courants

  • Aucune photo trouvée : Vérifiez que le dossier source contient des formats pris en charge (JPG, HEIC, RAW, etc.)
  • Ville incorrecte détectée : Téléchargez la base de données spécifique au pays via "Gérer la base de données pays"
  • L'application se bloque : Assurez-vous d'avoir au moins 200 Mo d'espace libre pour les opérations de base de données
